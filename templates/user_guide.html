eAI Ministry Tool User Guide

Introduction

Welcome to the eAI Ministry Tool, a comprehensive platform designed to assist ministry leaders and theological researchers.

This tool integrates several powerful modules to support your ministry work:

  • Doctrine Comparison: Compare theological positions across denominations
  • Sermon Preparation: Create, organize, and enhance sermon content
  • Pastoral Counseling: Manage counseling sessions with scriptural insights
  • Resource Library: Access and manage theological resources
  • Apologetics: Respond to objections and prepare for theological discussions

The platform is built with a particular focus on amillennial theology as a foundational perspective for comparing other theological viewpoints.

Getting Started

Account Creation and Login

  1. Visit the homepage and click "Register" to create a new account
  2. Fill in your username, email, and password
  3. Once registered, log in using your credentials
  4. After logging in, you'll be directed to your personalized dashboard

Dashboard Overview

Your dashboard provides quick access to all modules and displays recent activity. From here you can:

  • Access any of the five main modules
  • View recent sermons, comparisons, or resources you've created
  • See important notifications or system updates
  • Access your profile settings
Tip: Customize your profile to enhance your experience. Add denomination preferences to see more relevant content.

Doctrine Comparison Module

The Doctrine Comparison module allows you to compare theological positions across different denominations.

Key Features:

  • Standard Comparison: Compare any denominations side by side
  • Amillennial Foundation Comparison: Compare other theological perspectives against amillennial theology
  • Save Comparisons: Store comparison results for future reference
  • Topic Selection: Choose specific theological topics to compare

How to Use:

  1. Navigate to the Doctrine module from your dashboard
  2. Choose either "Standard Comparison" or "Amillennial Foundation Comparison"
  3. Select the denominations you wish to compare
  4. Select the theological topics of interest
  5. View the side-by-side comparison results
  6. Optionally save the comparison for future reference
Note: The Amillennial Foundation Comparison automatically includes amillennial theology as the baseline for comparison.

Sermon Preparation Tools

The Sermon module provides comprehensive tools for creating, organizing, and enhancing sermon content.

Key Features:

  • Sermon Builder: Create individual sermons with structured content
  • Series Planner: Organize multiple sermons into cohesive series
  • Outline Generator: Get suggested outlines based on scripture passages
  • Illustration Finder: Discover relevant illustrations for your sermons

Creating a Sermon:

  1. Navigate to the Sermon module and select "Build Sermon"
  2. Enter the sermon title, scripture passage, and theme
  3. Use the "Generate Outline" button to get a suggested structure
  4. Customize the outline points as needed
  5. Add illustrations using the "Find Illustrations" feature
  6. Enter your sermon content in the main editor
  7. Save your sermon when complete

Planning a Sermon Series:

  1. Select "Sermon Series" from the Sermon module
  2. Click "Create New Series"
  3. Enter the series title, description, theme, and date range
  4. Save the series
  5. Add existing sermons to the series or create new ones
  6. Arrange sermons in the desired order within the series
Tip: When creating a series, consider using a consistent theme or scripture book to maintain cohesion.

Pastoral Counseling Module

The Counseling module helps organize pastoral counseling sessions and provides scriptural guidance for various counseling situations.

Key Features:

  • Session Management: Create and track counseling sessions
  • Scripture Suggestions: Get relevant scripture passages for specific topics
  • Counseling Advice: Access guidance for common pastoral situations

Creating a Counseling Session:

  1. Navigate to the Counseling module and select "New Session"
  2. Enter the session title, description, and topic
  3. Use the "Suggest Scriptures" feature to find relevant passages
  4. Record session notes and action items
  5. Save the session for future reference
Important: All counseling sessions are private and only accessible to you. Ensure confidentiality is maintained.

Resource Library

The Resource Library provides access to theological materials from various sources, with a special emphasis on amillennial authors.

Key Features:

  • Resource Browser: Search and filter theological resources
  • Import Tool: Access materials from e-Sword, Logos, and other sources
  • Amillennial Collection: Special focus on amillennial authors and resources
  • Personal Contributions: Add your own resources to the library

Accessing Resources:

  1. Navigate to the Resources module
  2. Browse the library or use search/filter options
  3. Click on a resource to view its details
  4. Read content excerpts or follow links to external resources

Importing External Resources:

  1. Select "Import from e-Sword/Logos" in the Resources module
  2. Choose the external source (e-Sword, Logos, Amillennial authors)
  3. Select the resource type and topics of interest
  4. Review the available resources and select ones to import
  5. Complete the import process
Tip: Filter resources by "is_amillennial" to focus on materials from amillennial authors.

Apologetics Module

The Apologetics module helps you engage with common objections to amillennialism and prepare for theological discussions.

Key Features:

  • Objection Database: Common objections to amillennial theology
  • Response Generator: Get suggested responses to objections
  • Theological Authors: Access insights from key amillennial writers
  • Debate Preparation: Tools for preparing for theological discussions

Responding to Objections:

  1. Navigate to the Apologetics module
  2. Browse objections by category or search for specific topics
  3. Select an objection to view details
  4. Read existing responses or generate a new response
  5. Add your own responses if desired

Preparing for Debates:

  1. Select "Debate Preparation" in the Apologetics module
  2. Choose a theological category to focus on
  3. Review common objections and suggested responses
  4. Access key scriptures and talking points
  5. Generate additional responses as needed
Remember: The goal is to present amillennial theology clearly and respectfully, not to "win" debates.

Amillennial Theological Focus

The eAI Ministry Tool uses amillennialism as a foundational theological perspective throughout its various modules.

Key Amillennial Resources:

  • Historical Authors: Augustine, Martin Luther, John Calvin
  • Modern Authors: Geerhardus Vos, Anthony Hoekema, Sam Storms, Kim Riddlebarger
  • Key Texts: "The Bible and the Future" (Hoekema), "Kingdom Come" (Storms)

Amillennial Distinctives:

The following core beliefs of amillennialism are emphasized throughout the platform:

  • The millennium in Revelation 20 is symbolic of Christ's current reign
  • The Church is the spiritual fulfillment of Israel's promises
  • The Kingdom of God is "already but not yet" fully realized
  • A single general resurrection will occur at Christ's return
  • Old Testament prophecies are often fulfilled spiritually in Christ and the Church
Note: While amillennialism is the foundational perspective, the tool respects and accurately represents other theological viewpoints.
